# Donezilla - Task List Project  

![Donezilla Logo](./frontend/assets/donezilla-logo-full.png)  

A simple task management application with filtering capabilities.  

## Features  
- Add new tasks  
- Mark tasks as completed  
- Remove tasks  
- Filter tasks by:  
  - All tasks  
  - Active tasks  
  - Completed tasks  

## Technologies Used  
- React Native and Node.js
